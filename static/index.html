ZettelDrop

You have reached the backend services of ZettelDrop. This is not the location of the user interface. Currently there is no website user interface. The only way to interact with the service is to use a program like Postman that lets the user send GET and POST requests, etc.

post '/login'

@bodyParam: username

@bodyParam: password

post '/register'

@bodyParam: username

@bodyParam: password

post '/logout'

destroy session

get '/isloggedin'

returns 'true' if logged in with session cookie, false otherwise
