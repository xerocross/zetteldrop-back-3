ZettelDrop

You have reached the backend services of ZettelDrop. This is not the location of the user interface. Currently there is no website user interface. The only way to interact with the service is to use a program like Postman that lets the user send GET and POST requests, etc.

post '/b/login'

@bodyParam: username

@bodyParam: password

Logs into existing user account.

post '/b/register'

@bodyParam: username

@bodyParam: password

Creates a new user account.

post '/b/logout'

destroy session

get '/b/isloggedin'

returns 'true' if logged in with session cookie, false otherwise

post '/b/zettel'

@bodyParam: text

Creates a new zettel with the given text. Use #hashtags for tagging.

get '/b/zettel/:id'

Get Zettel with id :id for viewing.

post '/b/zettels'

@bodyParam: queryString

Searches for user zettels matching by queryString using the main ZettelDrop business logic.
